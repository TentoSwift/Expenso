# Budgety

家族や友人と支出を共有できる、イベント単位の家計管理iOSアプリ

## 概要

Budgetyは、旅行やイベント単位で複数人の支出を共有・管理できるiOSアプリです。
日々の支出をシンプルに記録するだけでなく、メンバー間の貸し借りを自動計算し、
最少回数で精算できる送金プランを提案します。

家計を「個人のもの」ではなく「家族や仲間と共有するもの」として扱える、
人と人とのつながりを支える家計簿を目指しています。

## 主な機能

- **イベント単位の家計管理**：旅行や飲み会など、目的別に支出を整理
- **iCloudによる複数人共有**：家族や友人とリアルタイムで家計データを共有
- **自動精算機能**：メンバー間の貸し借りを計算し、最少回数の送金プランを提案
- **多通貨対応**：海外旅行などで異なる通貨の支出も記録可能
- **AIによるカテゴリ自動提案**：入力した内容からカテゴリを自動推定
- **レシートOCR**：レシートを撮影して支出情報を自動入力
- **カテゴリ別フィルタリング・期間別集計**

## 技術スタック

- **言語**：Swift 5.x
- **UIフレームワーク**：SwiftUI
- **永続化**：Core Data
- **クラウド同期・共有**：CloudKit (NSPersistentCloudKitContainer / CKShare)
- **オンデバイスAI**：Apple Intelligence (Foundation Models)
- **OCR**：VisionKit
- **非同期処理**：Swift Concurrency (async/await)

## 設計上の工夫

### データ永続化方式の選定

Core Data・SwiftData・Realmを比較検討しました。
最終的にCore Dataを選択した理由は、複数人でのデータ共有機能を実装するうえで、
CloudKitとの統合（NSPersistentCloudKitContainer）と
共有機能（CKShare）によってプライバシーを担保して共有することができました。

### 精算アルゴリズム

支出と受益者の情報からメンバー間の債権債務を集計し、
送金回数を最小化する精算プランを提案する仕組みを実装しました。

### Apple Intelligenceによる入力支援

ユーザーが入力した支出内容（例：「ラーメン」）から、
オンデバイスのFoundation Modelsを用いてカテゴリ（例：「食費」）を提案します。
オンデバイス推論のためプライバシーを保ったままAI機能を活用できます。
